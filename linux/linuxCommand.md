   - crontab -e 编写定时脚本
   - 每五分钟执行一次： */5 * * * *
   - 每小时执行一次 ： 0 * * * *
   - 每天执行一次： 0 0 * * *
   - 每周执行一次： 0 0 * * 0
   - 每月执行一次： 0 0 1 * *
   - 每年执行一次： 0 0 1 1 *
```
crontab文件中的行由6个字段组成，不同字段间用空格或者tab键分割。前5个字段指定命令要运行的时间
分钟（0-59）
小时（0-28）
日期（1-31）
月份（1-12）
星期几（0-6，其中0代表星期日）
第六个字段是一个要在适当时间执行的字符串
```
